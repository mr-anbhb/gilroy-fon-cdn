# gilroy-fon-cdn
Using import:

```

@import url('https://raw.githubusercontent.com/mr-anbhb/gilroy-fon-cdn/main/gilroy_font_cdn.css');

font-family: 'gilroy';

```

or

```

<link rel="stylesheet" href="https://raw.githubusercontent.com/mr-anbhb/gilroy-fon-cdn/main/gilroy_font_cdn.css"/>

font-family: 'gilroy';

```
